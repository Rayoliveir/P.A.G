# P.A.G - Plataforma de Aprendizagem Gamificada
Uma plataforma de estudos com mecânicas de jogo (gamificação), onde o usuário completa desafios e ganha pontos ou medalhas.

## 🎯 Objetivo do Projeto
Criar uma aplicação web que simula uma plataforma de estudos, onde o usuário aprende por meio de desafios interativos e recebe recompensas conforme seu progresso. A ideia é usar mecânicas de gamificação para tornar o aprendizado mais envolvente

## 🛠️ Tecnologias Utilizadas

  - HTML: Estrutura das páginas (desafios, perfil do usuário, dashboard).
  - CSS: Estilização, temas visuais, animações e responsividade.
  - JavaScript: Lógica dos desafios, sistema de pontos, controle de progresso, interações com o usuário e salvamento no             localStorage.

## Diferencial 
Ensinar lógica de programação ou outro tema usando um jogo web.
